# Проект: TenChat

## Описание

Этот проект представляет собой простую чат-систему с возможностью общения между анонимными пользователями в реальном времени. Каждый пользователь получает случайное имя, и все сообщения отображаются в общем чате. Проект использует **Node.js** и **Socket.IO** для обработки сообщений в реальном времени.

## Функциональные возможности

- Анонимные пользователи (имена генерируются автоматически).
- Отправка и получение сообщений в реальном времени.
- Мгновенное обновление чата для всех пользователей.
- Чат доступен через веб-интерфейс.

## Технологии

- **Node.js**: серверная среда для запуска приложения.
- **Socket.IO**: библиотека для обмена сообщениями в реальном времени.
- **HTML/CSS**: фронтенд для отображения чата и модального окна с README.
